This plugin is add Google+, Twitter and Facebook Like buttons.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. If you need, go to 'Settings' -> 'Smart Social Share' to configure
